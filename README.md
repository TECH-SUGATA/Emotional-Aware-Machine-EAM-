## ✨ About the Project

**Emotional Aware Machine (EAM)** is a next-generation AI system that can detect, analyze, and interpret human emotions in real-time.

By combining **Computer Vision, Natural Language Processing, and Deep Learning**, this system enables machines to understand human feelings and respond intelligently.

> 💡 *“Not just artificial intelligence — emotional intelligence.”*

---

## ⚡ Core Features

* 🎭 **Facial Emotion Recognition** (Happy, Sad, Angry, etc.)
* 🎤 **Voice Sentiment Detection**
* 💬 **Text Sentiment Analysis (NLP)**
* 📊 **Real-time AI Dashboard**
* ⚡ **Fast & Responsive System**
* 🧠 **Multi-modal AI (Face + Voice + Text)**

---

## 🧠 AI Architecture

```
User Input (Face / Voice / Text)
            ↓
   Data Processing Layer
            ↓
   AI Models (CV + NLP + DL)
            ↓
   Emotion Classification
            ↓
   Output + Visualization Dashboard
```
